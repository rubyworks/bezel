# Bezel

> **Archived.** This was an experimental project to overcome the limitations of
> using different versions of the same package in the same Ruby process. It never
> reached a production-ready state. This repository is preserved for historical reference.

## Description

The idea of Bezel was to overcome the limitations of using different
versions of the same package in the same Ruby process. It worked by
evaluating package code into anonymous modules, allowing multiple
versions to coexist.
